Session Info

What you'll be listening to

Keynote - Adam Cogan

Topic TBA

Session 1 - Todd Whitehead

Azure Ninja Tips & Tricks

We all know Azure is a powerful platform but many aren’t aware of  the little features lurking in the corners than can transform you from an Azure Acolyte to full blown Azure Ninja. In this whirlwind session we’ll cover tips on everything from UI Customization to CLI’s lurking in unexpected places and from free tools and services to mysterious repositories of wisdom and enlightenment.

Session 2 - Adam Stephensen

Bots - the Next UI Revolution

The last decade saw an explosion in mobile applications. Today's customers expect an amazing mobile experience. Bots are the new UI frontier. Siri, Google Assistant and Alexa started the trend, but no-UI is the future of UI. Users are going to expect to be able to have conversations with companies and organisations in the client that they have on hand - their phone, a website they are on, Facebook messenger, via a phone call or even email or text message. Don't believe me ? Gartner predicts that by 2020 30% of http requests will be via bots. In this session we will explore how the Microsoft Bot Framework makes it easy to build and connect intelligent bots to interact with users and services. Take your existing applications, your FAQ or a great new idea you have and build a bot that will run from your website or from Cortana, Skype, Teams, Slack, Facebook Messenger, Skype and more.

If you missed the mobile revolution - why not lead in the bot wars ?

Session 3 - Alessandro Cardoso

Identity Modernization With Azure

In the 2017 Gartner Magic Quadrant for Access Management (AM MQ), Azure Active Directory is placed in the “leaders” quadrant, which validates Microsoft vision of providing a complete identity solution for supporting employees, partners and customers all backed by world class security. In this session we will be covering the following interactions, gathered from engagements with Financial Services customers, providing recommended approaches for Azure AD, Azure B2B and Azure B2C:

  • Organization retains full ownership of customer experience
  • Customer ownership is shared between Organization and Partner/Customer organization
  • Partner/Customer organization takers ownership of their customer experience

Session 4 - Dan Toomey

Serverless iPaaS in Azure

This session will explore how to build powerful event-driven integration solutions using the serverless platform offerings in Azure, including Logic Apps, Azure Functions, Service Bus, and Event Grid.

Session 5 - Elliot Munro

Automate Office 365 Administration with Azure Functions and PowerShell

Learn how to use Azure Functions, PowerShell and Microsoft Graph to automate manual tasks, secure environments, apply consistent policies and get important alerts. This session is aimed at IT Pros who want to see how far they can get with a little powershell skill and the power of Azure Functions. It will include a demonstration of creating a user offboard script with an approval process using Azure Functions, Microsoft Flow Approvals and SharePoint Online. It will also include examples of existing admin scenarios that can be automated using HTTP, Queue and Timer Triggered Azure Functions.

Session 6 - David Alzamendi

Ingesting, Processing & Displaying Real-Time Information with Azure

Have you been asked about displaying real time information? What about ingesting millions of records per second coming from IoT or devices?
To answer these questions, Azure offers different platforms such as Azure IoT Hubs, Azure Event Hub or even Apache Kafka. From a data analytics point of view, I will go through a variety of characteristics, scenarios and differences that we need to take into account in order to choose the best fit for our solution.
